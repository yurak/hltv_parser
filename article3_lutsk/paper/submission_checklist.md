# Submission Checklist — журнал №85 (ЛНТУ)

Звірка перед поданням. Джерело вимог: `paper/journal_requirements.md`.

## Формат файлу
- [ ] MS Word `.docx`, A4
- [ ] Сторінки НЕ нумеровані
- [ ] Поля: верх/низ 1,5 · ліво 2,5 · право 2 см (дзеркальні); колонтитули 1,25 см
- [ ] TNR: текст 11 пт, анотації 9 пт, шапка 11 пт; інтервал одинарний
- [ ] Абзац: текст 1,25 см, анотації 1 см
- [ ] Формули — редактор формул MS Word
- [ ] Обсяг 5–10 сторінок (≤10, щоб уникнути доплати)
- [ ] Завантажено й застосовано офіційний Word-шаблон з cit.lntu.edu.ua

## Структура (IMRAD)
- [ ] УДК
- [ ] ORCID усіх авторів
- [ ] Автори: ПІБ, ступінь, місце роботи
- [ ] Назва — укр + англ
- [ ] Анотація 200 слів — укр + англ
- [ ] Ключові слова 5–10 — укр + англ
- [ ] Вступ → Матеріали і методи → Результати → Обговорення → Висновки
- [ ] Список бібліографічного опису (мовою оригіналу)
- [ ] References — повний романізований

## Бібліографія
- [ ] Стиль ДСТУ 8302:2015
- [ ] Числові посилання [1], [2-3]
- [ ] Транслітерований блок References
- [ ] Перевірено через grafiati.com

## Якість і доброчесність
- [ ] Перевірка на плагіат пройдена
- [ ] Кожне число в Results має provenance (outputs/)
- [ ] Немає необґрунтованих причинних тверджень

## Подання
- [ ] Email на cit@lntu.edu.ua із зазначенням напряму
- [ ] Подання через OJS
- [ ] Подано до дедлайну (найближчий: 20.08.2026 → вересневий випуск)

## Оплата (уточнити в редакції)
- [ ] З'ясовано APC з урахуванням співавторства (Y. Furgala) — пільга одноосібного авторства, ймовірно, не діє
- [ ] Вирішено питання DOI (+60 грн)
