## СЛАЙД 5: Методологія — загальний підхід

### Чому розміри ефектів, а не p-values?

**На слайді:**

❌ **Проблема p-values:**
- При n > 3000 навіть мінімальні відмінності "статистично значущі"
- Не показують практичну важливість
- Залежать від розміру вибірки

✅ **Рішення — розміри ефектів:**
- η² (eta-squared) — частка поясненої дисперсії
- Cohen's d — стандартизована різниця середніх
- Масштабно-незалежні міри

**Текст для озвучення:**

Перш ніж перейти до конкретних методів, поясню загальний підхід. При великих вибірках, як наша, традиційні p-values стають практично марними — навіть мізерні відмінності будуть "статистично значущими". Тому ми використовуємо розміри ефектів — міри, які показують практичну значущість відмінностей незалежно від розміру вибірки.

## СЛАЙД 6: Методологія — аналіз по картах

### ANOVA з Eta-squared (η²)

**На слайді:**

**Метод:** Однофакторний дисперсійний аналіз (ANOVA)

**Формула:**
```
η² = SS_between / SS_total
```

**Інтерпретація η²:**
| Значення | Розмір ефекту |
|----------|---------------|
| < 0.01 | Незначний |
| 0.01 - 0.06 | Малий |
| 0.06 - 0.14 | Середній |
| > 0.14 | Великий |

**Поріг інваріантності:** η² < 0.06

**Текст для озвучення:**

Для аналізу відмінностей між вісьмома картами ми використали однофакторний дисперсійний аналіз ANOVA з розміром ефекту eta-squared. Eta-squared показує, яка частка загальної дисперсії ознаки пояснюється відмінностями між картами.

Згідно з загальноприйнятими критеріями, значення менше 0.06 вказує на малий ефект. Ми обрали цей поріг для визначення інваріантних ознак — тих, де карта пояснює менше 6% дисперсії.

## СЛАЙД 7: Методологія — аналіз по версіях

### Mann-Whitney U з Cohen's d

**На слайді:**

**Метод:** Критерій Манна-Уітні (непараметричний)

**Чому непараметричний тест?**
- Ігрова статистика має скошені розподіли
- Не потребує припущення нормальності
- Робастний до викидів

**Формула Cohen's d:**
```
d = |μ₁ - μ₂| / σ_pooled
```

**Поріг інваріантності:** d < 0.5

**Текст для озвучення:**

Для порівняння CS2 та CSGO — двох незалежних груп — ми застосували критерій Манна-Уітні. Це непараметричний тест, який не потребує припущення про нормальний розподіл даних. Це важливо, оскільки ігрова статистика часто має скошені розподіли з викидами — є багато середніх гравців і небагато елітних.

Як міру розміру ефекту ми використали Cohen's d. Значення менше 0.5 вказує на малий-середній ефект, що ми вважаємо порогом інваріантності.

## СЛАЙД 8: Методологія — аналіз по сторонах

### Парний t-тест з Cohen's d

**На слайді:**

**Метод:** Парний t-тест для залежних вибірок

**Чому парний тест?**
- Кожен гравець має статистику і за CT, і за T сторону
- Дані природно парні
- Враховує внутрішньосуб'єктну кореляцію

**Формула для парного Cohen's d:**
```
d = |μ_diff| / σ_diff
```

**Поріг інваріантності:** d < 0.5

**Текст для озвучення:**

Для порівняння статистики за терористів та контр-терористів ми використали парний t-тест. Це принципово, оскільки кожен гравець має показники за обидві сторони з тих самих матчів — це природно парні дані. Парний тест дозволяє елімінувати дисперсію від індивідуальних відмінностей між гравцями та зосередитись саме на ефекті сторони.

   - Не аналізуємо стабільність ознак у часі для окремих гравців

4. **Ролі:** Не враховано ігрові ролі
   - AWPer vs Rifler vs IGL можуть мати різні патерни

**Текст для озвучення:**

Важливо згадати обмеження нашого дослідження. По-перше, вибірка включає лише топ-гравців світу — результати можуть відрізнятись для аматорського рівня, де варіативність стилів гри більша.

По-друге, ми працювали з агрегованою статистикою по матчах, що не дозволяє аналізувати динаміку на рівні окремих раундів.

По-третє, це cross-sectional аналіз — ми не досліджували, чи залишаються ознаки стабільними для конкретного гравця протягом кар'єри.

Нарешті, ми не враховували ігрові ролі — AWPer та rifler мають принципово різні профілі, що може впливати на інваріантність.

## ДОДАТОК: Технічні деталі для Q&A

### Часті питання та відповіді:

**Q: Чому поріг η² = 0.06, а не 0.01?**
A: Поріг 0.01 (незначний ефект) занадто суворий для практичних цілей. Малий ефект (0.01-0.06) вже вказує на практично незначущі відмінності. Ми обрали верхню межу малого ефекту як консервативний поріг.

**Q: Чому для сторін парний тест, а для версій — ні?**
A: Для сторін дані природно парні — той самий гравець грає за обидві сторони в кожному матчі. Для версій дані незалежні — це різні матчі, часто різні гравці на різних етапах кар'єри.

**Q: Чи враховано мультиколінеарність ознак?**
A: Так, ми видалили очевидно похідні ознаки (наприклад, total_kills якщо є kills_per_round). Однак деяка кореляція залишається — це можна врахувати в майбутніх дослідженнях через факторний аналіз.

**Q: Чому rating 3.0 інваріантний, якщо він композитний?**
A: Rating 3.0 розроблений HLTV саме для нівелювання контекстних факторів. Наші результати валідують цей підхід — rating справді залишається стабільним.

**Q: Як щодо meta-shifts у CS2?**
A: Наші дані охоплюють період після стабілізації CS2 мети. Для перших місяців після релізу результати можуть відрізнятись.
