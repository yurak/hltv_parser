## Анотація (чернетка)

Робота присвячена задачі ідентифікації поведінкових шаблонів користувачів онлайн-сервісів на прикладі фінансових транзакцій із використанням банківських платіжних карток. У попередньому дослідженні [1] авторами було показано, що при коректному виборі сімейств ознак типові ролі користувачів Counter-Strike 2 утворюють окремі області у багатовимірному просторі ознак, які не зводяться до одного числового індексу. У цій роботі пропонується розширення зазначеної методології на клас фінансових онлайн-сервісів. Побудовано простір поведінкових ознак транзакцій, що містить сімейства часових, швидкісних (velocity), структурних за сумою та ідентифікаційних (цільовокодовані) ознак. На публічному наборі даних Predict Chargeback Frauds (Kaggle, 11 127 транзакцій, травень 2015) продемонстровано, що шахрайські транзакції утворюють статистично відокремлений область у просторі ознак (MANOVA, p<0.001), а розділюваність досягається переважно за рахунок швидкісних, а не часових, ознак. Отримані висновки підтверджуються на додатковому наборі IEEE-CIS Fraud Detection (~590 тис. транзакцій). Отриманий простір ознак забезпечує якість класифікації PR-AUC = ..., ROC-AUC = ..., при точності серед 1 % найбільших = ..., що підтверджує практичну значущість ідентифікованих поведінкових шаблонів.

(чорнові значення метрик заповнюються після експериментів)

## Структура статті

### 1. Вступ (приблизно 1 стор.)
- Актуальність: масштаб втрат від карткового шахрайства; світова та українська статистика (НБУ, УАФК).
- Проблема: антифрод-системи використовують десятки ознак без систематичного теоретичного обгрунтування; відсутня уніфікована типологія поведінкових ознак.
- Наукова теза: ідентифікація поведінкових шаблонів користувачів як задача pattern recognition у багатовимірному просторі ознак, методологія якої демонструвалася авторами раніше на прикладі Counter-Strike 2 [1].
- Мета та задачі. Наукова новизна. Практична значущість.

### 2. Аналіз літературних джерел (приблизно 1.5 стор.)
- Класичні підходи до виявлення шахрайства: Dal Pozzolo, Carcillo, Bahnsen, Lucas, Jurgovsky.
- Методи pattern recognition для транзакційних даних: PCA, clustering, separability analysis.
- Відмінність запропонованого підходу: фокус на ідентифікації поведінкових шаблонів як областів простору ознак, а не на побудові чорно-скринькової моделі.

### 3. Матеріали та методи (приблизно 2.5 стор.)

**3.1. Характеристика наборів даних.**

*Основний датасет:* Predict Chargeback Frauds (Kaggle, автор `dmirandaalves`, 2019) — 11 127 транзакцій, часовий діапазон 01.05.2015–30.05.2015, частка шахрайських транзакцій (Chargebacks, CBK) 5.14%, чотири вихідні поля: маскований номер картки, дата-час транзакції, сума, мітка CBK (Yes/No). Цей набір обраний основним з двох причин: (а) його мінімальна схема вимагає побудови простору ознак з нуля, що відповідає фокусу статті на інженерії ознак; (б) масштаб набору співмірний з масштабом попередньої роботи авторів [1], що забезпечує послідовність методологічного аналізу.

*Валідаційний датасет:* IEEE-CIS Fraud Detection (Kaggle / Vesta Corporation, 2019) — близько 590 тис. реальних транзакцій e-commerce, часовий діапазон ~6 міс., ~3.5% шахрайських. Використовується у заключному підтверджувальному експерименті для перевірки переносимості методології на датасет іншого масштабу і ринку.

**3.2. Побудова простору ознак.** Таксономія чотирьох сімейств ознак:
- **Часові ознаки:** година доби (sin/cos), день тижня, вихідні, нічний сегмент.
- **Швидкісні (velocity) ознаки:** інтервал часу від попередньої транзакції картки (gap_min_prev); кількість транзакцій картки за вікнами 1 год / 6 год / 24 год; сумарний обсяг за тими ж вікнами; кумулятивна кількість транзакцій картки.
- **Структурні ознаки суми:** логарифм суми, індикатори подільності на 10/50/100, наявність копійчаної частини, пороги 200/500 USD.
- **Ідентифікаційні (цільовокодовані) ознаки:** згладжений за Байєсом fraud rate BIN-а та значення суми (за допомогою k-smoothing, розрахунок на тренувальній частині).

Таксономія сумісна зі структурою попередньої роботи [1], де використовувались сімейства: базові HLTV-ознаки, side-specific, map-dependent. Теза: структурування ознак у типологічні сімейства — універсальний прийом pattern recognition у поведінкових даних онлайн-сервісів.

**3.3. Методи аналізу простору ознак.**
- Кореляційний аналіз сімейств ознак (Pearson, Spearman).
- PCA для візуалізації та оцінки внутрішньої розмірності.
- ANOVA / MANOVA для перевірки статистичної значущості розділюваності груп (легітимні vs шахрайські).
- Silhouette-індекс для кількісної оцінки відокремленості.
- Permutation importance для ідентифікації домінантних ознак у розділенні груп.

**3.4. Класифікатори як кількісна верифікація розділюваності.** Логістична регресія (як калібрований baseline) та HistGradientBoostingClassifier (як нелінійний baseline). Часо-впорядкований (time-based) поділ train/test 75/25 для уникнення витоку майбутнього.

**3.5. Метрики.** ROC-AUC, PR-AUC (average precision), precision/recall при top-k (k ∈ {1, 3, 5, 10} %), confusion matrix на фіксованому порозі.

### 4. Результати (приблизно 3 стор.)

**Експеримент E1.** Описова статистика груп у кожному сімействі ознак. Таблиця середніх, медіан, міжквартильних розмахів для легітимних та шахрайських транзакцій.

**Експеримент E2.** PCA-розклад простору ознак. Візуалізація перших двох головних компонент. Очікуваний результат: шахрайські транзакції утворюють видимо відокремлений область. Зафіксувати частку дисперсії, збережену першими двома і п'ятьма компонентами.

**Експеримент E3.** MANOVA / ANOVA для перевірки розділюваності. Для кожного сімейства ознак окремо та для об'єднаного простору. Звітуються λ Вілкса, F-статистика, p-значення.

**Експеримент E4.** Ablation-аналіз сімейств ознак. Таблиця:
| Конфігурація | ROC-AUC | PR-AUC | точності серед 1 % найбільших |
|---|---|---|---|
| Повний простір ознак | ... | ... | ... |
| Без часових ознак | ... | ... | ... |
| Без швидкісних ознак | ... | ... | ... |
| Без структурних ознак | ... | ... | ... |
| Без ідентифікаційних ознак | ... | ... | ... |
Гіпотеза: видалення швидкісного сімейства призводить до найбільшої втрати якості; видалення часового — до мінімальної.

**Експеримент E5.** Permutation importance за PR-AUC — рейтинг ознак за внеском у розділення груп.

**Експеримент E6.** Перевірка опосередкування часових ознак швидкісними. Кореляційний аналіз + регресія часових ознак на швидкісні; аналіз залишків.

### 5. Обговорення (приблизно 1 стор.)
- Інтерпретація PCA та ANOVA: шахрайська поведінка — окремий область простору ознак, що узгоджується з результатом для ігрових ролей у [1].
- Практичні висновки для проектування антифрод-систем.
- Обмеження: аналіз побудований на даних одного ринку (e-commerce, Vesta), для повного підтвердження доменно-агностичності необхідне розширення на додатковий фінансовий датасет.

### 6. Висновки (0.5 стор.)
- Підтверджено методологічну наступність з [1].
- Показано, що шахрайська поведінка у просторі поведінкових ознак є окремим областьом.
- Ідентифіковано домінантне сімейство ознак (очікувано — швидкісне).
